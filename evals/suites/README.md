# Local suites

Suite notes describe bounded owner-local invariants. A note is not executable;
only a validated adjacent execution sidecar may route an owner-run trial.

- `session-memory-skill-routing`: deterministic discovery, compatibility,
  composition, portability, and attribution-contract pressure.
- `session-memory-skill-behavior`: real public-safe fixture execution across
  critical capability branches inside an ephemeral behavioral sandbox.
