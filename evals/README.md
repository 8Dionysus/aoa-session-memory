# Local eval port

This active port carries bounded proof pressure for the session-memory skill
system. The routing suite covers deterministic routing, negative
applicability, coexistence, typed composition, package portability, and
attribution contracts. The behavioral-sandbox suite executes public-safe
fixtures across every critical capability branch with ephemeral filesystem,
network-denial, preview/apply, recovery, and source-mutation guards.

Behavioral no-skill/current/candidate model trials and their raw traces remain
session-local until `aoa-evals` accepts an appropriate central proof surface.

Passing the local suite is not a skill-benefit verdict.

`aoa-evals` owns verdict, scoring, regression, and proof doctrine authority.
