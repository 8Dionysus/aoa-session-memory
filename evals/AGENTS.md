# AGENTS.md

## Role

This local port preserves public-safe session-memory skill-routing,
composition, portability, and attribution pressure. It does not own proof
doctrine, scoring, regression truth, promotion, or final benefit verdicts.

## Boundaries

- `aoa-evals` owns the shared local-port contract, verdict, scoring,
  regression, and proof doctrine authority.
- Local suites may execute only through a reviewed sidecar, just-in-time
  validation, environment capture, and an execution receipt.
- Raw transcripts, private sessions, one-off prompts, and model judgments do
  not enter this directory.
- A green deterministic suite proves only its named source invariant.

## Validation

Use the pinned `aoa-evals` local-port validator, then run the exact reviewed
suite sidecar only through the selected owner/apply route.
